Fichier HTML
  • Les images doivent être sur votre serveur avec des liens absolus pointant sur vos images de vos serveurs.
  • Balise (IMG SRC=): l’adresse doit être obligatoirement en double guillemets (“). Le simple guillemet (’) est non supporté.
  • Critère de compatibilité avec Microsoft Word 2007 :
    • Pas d’image d’arrière-plan
    • Pas de JavaScript
  • Balise (Tag) « MAP » supportée
  • Aucun CSS externe
  • Le CSS 2 et CSS 3 ne sont pas supportés
  • Utiliser des tableaux HTML
  • Ne pas utiliser les balises DIV
  • Utiliser l’attribut HTML “align” plutôt que “float”
  • Ne pas utiliser des images blanches pour créer des espaces
  • Javascript et Flash ne sont pas supportés, de même que les vidéos
  • Éviter les formats d’image PNG, BMP et TIF
  • Enlever balise HEAD et tous les éléments à l’intérieur de cette balise
  • Enlever attributs de la balise BODY (sans attributs)
